- [ ] Bảng phân tích, thiết kế nội dung đề tài

## 3. Quy Định Trình Bày (Formatting Rules)

### 3.1. Giấy và Canh Lề

| Thuộc tính          | Giá trị                            |
| ------------------- | ---------------------------------- |
| **Khổ giấy**        | A4 (21 × 29,7 cm), trắng, in 1 mặt |
| **Lề trái**         | 3,5 cm                             |
| **Lề phải**         | 2,0 cm                             |
| **Lề trên / dưới**  | 2,5 cm                             |
| **Header / Footer** | 1,0 cm                             |

> [!IMPORTANT]
> Không ghi tên đề tài, tên SV, tên GVHD ở header/footer.

### 3.2. Font Chữ và Cách Dòng

| Thuộc tính                           | Giá trị                                                |
| ------------------------------------ | ------------------------------------------------------ |
| **Kiểu chữ**                         | Times New Roman                                        |
| **Cỡ chữ nội dung**                  | 13                                                     |
| **Cỡ chữ tên đề tài (bìa)**          | 20, in đậm                                             |
| **Cỡ chữ tiêu đề trường/khoa (bìa)** | 14, in đậm                                             |
| **Thụt đầu dòng (tab)**              | 1,0 cm                                                 |
| **Cách dòng (line spacing)**         | 1,2 (nội dung chính) / 1,0 (TLTK, bảng, hình, phụ lục) |
| **Paragraph spacing**                | Before: 6 pt, After: 0 pt (giữa các mục)               |

### 3.3. Đánh Số Chương và Mục

```
CHƯƠNG 1              ← Chữ hoa, đậm, cỡ 14, giữa trang, bắt đầu trang mới
TÊN CHƯƠNG            ← Chữ hoa, đậm, cỡ 14, giữa trang

1.1 MỤC CẤP 1         ← Chữ hoa, đậm, sát lề trái
    1.1.1 Mục cấp 2   ← Chữ thường, đậm, cách lề 1,0 cm
    1.1.1.1 Mục cấp 3 ← Chữ thường, đậm, cách lề 1,0 cm
```

> [!NOTE]
> Sau tên chương và mục **không dùng** dấu chấm hay dấu hai chấm.

### 3.4. Đánh Số Trang

| Phần                              | Kiểu đánh số                     |
| --------------------------------- | -------------------------------- |
| Lời cảm tạ → Danh mục từ viết tắt | Số La-mã thường: i, ii, iii, ... |
| Chương 1 → Phụ lục                | Số Á-rập: 1, 2, 3, ...           |
| Vị trí                            | Giữa trang, cỡ 13                |

## 5. Quy Tắc Bảng, Hình, Công Thức

### Bảng

- **Đánh số:** `Bảng X.Y` (X = chương, Y = thứ tự). VD: `Bảng 3.1`
- **Vị trí tên:** Phía **trên** bảng, **canh trái**, cỡ chữ 13
- **Nguồn trích dẫn:** Phía **dưới** bảng, **canh giữa**, _in nghiêng_, cỡ chữ 11
- Không cắt bảng qua 2 trang. Nếu bảng dài, trang kế chỉ cần lặp tên cột
- Số thập phân dùng dấu phẩy (`,`), hàng ngàn dùng dấu chấm (`.`)

### Hình

- **Đánh số:** `Hình X.Y`. VD: `Hình 2.3`
- **Vị trí tên:** Phía **dưới** hình, **canh giữa**, cỡ chữ 13
- **Nguồn trích dẫn:** Phía dưới tên hình, _in nghiêng_, cỡ chữ 11
- Hình nên để chế độ **In line with text**, canh giữa, không làm khung

### Công thức

- Đánh số trong ngoặc đơn, đặt bên **lề phải**: `(1.1)`, `(1.2)`
- Giải thích ký hiệu khi xuất hiện lần đầu

## 6. Quy Tắc Trích Dẫn

| Trường hợp                 | Cách ghi                                                              |
| -------------------------- | --------------------------------------------------------------------- |
| **1 tác giả**              | Nguyễn Văn A (2009, trang 25) … hoặc … (Nguyễn Văn A, 2009, trang 25) |
| **2 tác giả**              | Hoàng Trọng và Chu Nguyễn Mộng Ngọc (2008, trang 76)                  |
| **≥ 3 tác giả**            | Võ Thành Danh và cộng sự (2005) / Henderson et al. (1987, p.64)       |
| **Cùng tác giả, cùng năm** | Thêm a, b, c: (2005a, 2005b)                                          |

### Tài Liệu Tham Khảo

- Xếp riêng theo ngôn ngữ (Việt, Anh, ...)
- **Tiếng Việt:** ABC theo **tên** (giữ nguyên thứ tự họ tên)
- **Tiếng Anh:** ABC theo **họ** (surname)
- Từ dòng thứ 2 trở đi: lùi vào 1 cm

**Mẫu — Sách:**

```
Nguyễn Anh Tuấn và Trần Ngọc Hải (1992), Kỹ thuật nuôi tôm sú.
    Nhà xuất bản Nông nghiệp, Hà Nội. 68 trang.
```

**Mẫu — Website:**

```
Min, K. (1998), Wastewater pollution in China.
    http://www.example.com, accessed on 17/3/2008.
```

**Mẫu — Tạp chí:**

```
Ho, Y.W. and S.S.Y. Nawawi (1969), Effects of carbon…
    Journal of Molecular Biology, 45: 567-575.
```
